# Mobility Platform

Multi-tenant NDIS-compliant mobility equipment rental platform with:

- Public booking site (React)
- Multi-location inventory
- Stripe + NDIS invoicing
- Driver PWA / Native iOS planned
- SMS/Email notifications
- PDF invoices
- Xero export
- Delivery fees by postcode
- Equipment maintenance logs
- ISO 27001 baseline security
